## Easter

Given a year, finds out the date of Easter Sunday for that year.  
Utilises formula at https://www.drupal.org/node/1180480

## Usage

```bash
$ gcc easter.c -o easter
$ ./easter
Enter Year: 2017
Easter is April 16 in 2017
```

## Installation

Compile with `gcc easter -o easter`
